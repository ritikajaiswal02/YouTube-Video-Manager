# YouTube Video Manager 🎬  
A simple Python project using Tkinter to manage YouTube video data.  

## Features  
✅ Add, update, and delete video records  
✅ Uses JSON file storage  
✅ Simple GUI with Tkinter  

## How to Run  
1. Install Python  
2. Run `python youtubeGui.py`
